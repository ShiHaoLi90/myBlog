# vuepress搭建及github站点部署

##项目搭建

# 1.全局安装yarn

yarn是NPM的替代品，也是js包管理工具，本教程以yarn包管理工具进行部署。 命令如下：

npm install yarn -g

验证是否安装成功,成功后显示版本，不成功则提示命令不存在

 yarn -v

# 2.更改yarn配置（可选）

如果您的网络环境较差的情况下，可以修改yarn的配置，使用淘宝镜像。

yarn config set registry https://registry.npm.taobao.org

# 3.创建vuepress

使用如下命令

yarn create vuepress

如果安装较慢，或者安装时网络连接不上，进行第2步操作

# 4.配置选择

除了选择是创建博客还是文档，其他无关紧要的配置可以随意填写 具体如图 ; ;

# 5.运行刚创建的项目

yarn dev

如果第4步选择的是blog时，直接运行yarn dev 会报错，解决办法是安装插件

yarn install

# 6.查看vuepress项目

打开本地浏览器输入：localhost:8080或者127.0.0.1：8080查看效果

# github站点发布

# 1.在github上创建一个仓库

; 1.仓库名字 2.初始化readme.md文件 3.支持MIT许可

# 2.项目根目录下创建一个.sh文件

将需要的常用脚本写入到sh文件当中，后续执行sh文件，自动运行脚本

#!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
yarn build

# 进入生成的文件夹
cd blog/.vuepress/dist

git init
git add -A
git commit -m 'deploy'


# 如果发布到 https://<USERNAME>.github.io/<REPO>
git push -f git@github.com:xx/myBlog.git master:gh-pages
# 链接远程仓库
git remote add origin https://github.com/xx/myBlog.git
#拉取远程仓库的文件
git pull --rebase origin master
#同步更新代码
git push -u origin master
cd -

# 3.执行sh文件

在xx.sh文件所在的目录下，鼠标右键，选择git bash here打开git bash 命令行执行sh文件

sh xx.sh

# 4.查看发布站点

进入github》仓库》settings》github pages ;
